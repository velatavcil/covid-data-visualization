# COVID-19 Türkiye İllere Göre Veri Analizi

Bu projede, Pandas ve Matplotlib kütüphaneleri kullanılarak Türkiye'nin 81 ili için COVID-19 verileri analiz edilmiştir. Veri seti `covid_turkiye_81_il.csv` dosyasından alınmıştır.

## İçerik

- Verilerin yüklenmesi ve incelenmesi
- Eksik verilerin kontrolü
- İllere göre vaka sayılarının görselleştirilmesi
- Türkiye haritası üzerine il bazlı vaka sayısı dağılımı (isteğe bağlı)
- Grafikler ile karşılaştırmalı analizler

## Kullanılan Teknolojiler

- Python
- Pandas
- Matplotlib
- NumPy (gerekiyorsa)
